## Configure backup policy
1. Navigate to Policies.
2. Define scope and schedule.
3. Save.

## Run backup
1. Open Backup page.
2. Select tenant.
3. Click Run Now.

## Restore data
1. Open Restores.
2. Choose backup.
3. Provide target path.
4. Submit.

Verification: Files appear in target path.
Rollback: Re-run backup with previous settings.
